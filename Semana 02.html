Semana 02

Aquí te voy a contar todo de la semana 02 sobre
Algunas Innovaciónes en soluciones TECH, ODS y como Utilizar Visual Studio Code.

Introducción

En las clases presenciales, iniciamos con dinámicas para generar propuestas innovadoras que aborden los ODS, mientras que en el entorno virtual comenzamos a adquirir conocimientos sobre cómo desarrollar nuestra bitácora digital.

Clase Presencial

En la sesión de hoy, dedicamos un tiempo considerable a analizar los Objetivos de Desarrollo Sostenible (ODS) establecidos por las Naciones Unidas. El propósito fue profundizar en los desafíos específicos que plantea cada objetivo, con el fin de sensibilizar a los estudiantes sobre la urgencia y la importancia de estos retos globales. Este entendimiento es crucial para fomentar una mentalidad orientada hacia la solución de problemas significativos a través de la innovación y la tecnología.

Para hacer la sesión más interactiva y práctica, propusimos una actividad creativa en la que los estudiantes fueron invitados a concebir una idea, ya sea tecnológica o no tecnológica, que pudiera contribuir a la resolución de al menos dos ODS. Esta tarea no solo buscaba estimular el pensamiento crítico y la creatividad, sino también incentivar la colaboración entre los estudiantes, quienes podían formar equipos para mezclar diversas habilidades y perspectivas.

Los estudiantes se mostraron entusiastas y participaron activamente en la brainstorming session. Utilizaron herramientas como mapas mentales y diagramas de flujo para organizar sus pensamientos y analizar cómo diferentes soluciones podrían abordar múltiples problemas de forma simultánea. Algunas propuestas incluyeron aplicaciones móviles para mejorar la educación en áreas remotas, sistemas innovadores para la gestión de residuos en ciudades, y proyectos de agricultura urbana para combatir el hambre y promover la sostenibilidad ambiental.

Clase Virtual

Para expandir y dar más detalle a tu texto, se puede describir con más profundidad la formación del grupo, la introducción al uso de Visual Studio Code y cómo esto se integra con GitHub, así como la importancia de comprender las estructuras de código. Aquí tienes un ejemplo: Durante la reciente sesión del curso, dimos un importante paso adelante al formar los grupos de trabajo que permanecerán juntos a lo largo de toda la experiencia de aprendizaje. Este proceso se llevó a cabo con el objetivo de fomentar la colaboración y el intercambio de habilidades entre los estudiantes, aspectos fundamentales en el ámbito del desarrollo de software.

Una vez formados los grupos, procedimos a instalar y configurar una herramienta esencial para cualquier desarrollador: Visual Studio Code. Este poderoso editor de código es preferido por muchos debido a su interfaz intuitiva y la capacidad de ser ampliado mediante extensiones que facilitan y optimizan la codificación en diversos lenguajes de programación. Durante esta etapa, los estudiantes exploraron algunas de las características más útiles de Visual Studio Code, incluyendo la función de "commit", que es vital para el control de versiones. Aprender a realizar commits permite a los desarrolladores guardar versiones del código de manera que puedan rastrear y revertir cambios si es necesario, una habilidad crucial cuando se trabaja en proyectos de software en equipo.

Además, integramos Visual Studio Code con GitHub, la plataforma líder para el alojamiento de código fuente que utiliza Git para el control de versiones. Mediante esta integración, los estudiantes aprendieron a conectar sus entornos locales de desarrollo con un repositorio remoto, lo que facilita la colaboración y el seguimiento de cambios en proyectos compartidos. Finalmente, nos centramos en la comprensión de las estructuras básicas del código, lo que permitió a los estudiantes empezar a editar y manipular código por sí mismos. Este conocimiento es fundamental para cualquier programador, ya que proporciona la base sobre la cual se construirán todas las habilidades de desarrollo futuro. Este conjunto de habilidades adquiridas no solo equipa a los estudiantes con las herramientas necesarias para su desarrollo académico y profesional, sino que también sienta las bases para su éxito en proyectos futuros, fomentando un ambiente de aprendizaje activo y aplicado.
