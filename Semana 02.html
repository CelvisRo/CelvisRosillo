Semana 02

Aquí te voy a contar todo de la semana 02 sobre
Algunas Innovaciónes en soluciones TECH, ODS y como Utilizar Visual Studio Code.

Introducción

En las clases presenciales, iniciamos con dinámicas para generar propuestas innovadoras que aborden los ODS, mientras que en el entorno virtual comenzamos a adquirir conocimientos sobre cómo desarrollar nuestra bitácora digital.

Clase Presencial

En esta sesión, se logró detallar las ODS para comprender los desafíos asociados a cada una y así inspirar a los estudiantes a investigar soluciones existentes globalmente que aborden estos problemas. Durante la clase realizamos una actividad de hacer una crear una idea tecnólogia o no técnologica que ayuda a resolver mas de 2 ODS, el ganador se llevaría un punto en el siguiente trabajo.

Clase Virtual

Formamos el grupo con el vamos a trabajar durante el curso. Descargamos el Visual Studio Code, conociendo algunas de sus herramientas, entre ellas el commit. Enlazamos con el Gitgub y aprendimos a reconocer las estructuras del código para comenzar a editar.
